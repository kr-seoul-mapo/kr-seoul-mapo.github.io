# 마포구 · 서울특별시

> 마포구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-seoul-mapo.github.io/**

마포구는 한강 북안에 자리한 서울의 자치구로, 옛 마포나루의 물류 전통 위에 오늘날 청년 문화와 미디어 산업이 겹쳐 성장한 곳이다. 홍대와 합정, 연남동 일대는 인디 음악과 카페, 출판·디자인 스튜디오가 밀집해 서울에서 가장 젊은 거리로 꼽힌다. 상암동 디지털미디어시티에는 지상파 방송사와 IT 기업이 모여 수도권 방송·콘텐츠 산업의 한 축을 이룬다. 경의선 숲길과 월드컵공원이 촘촘한 도심 사이로 녹지의 여유를 더한다. 강 건너 여의도·영등포와 마주 보며 서울 서북부의 관문 역할도 겸한다.
